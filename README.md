# Portfolio Website

A modern, responsive personal portfolio built with HTML, CSS, and JavaScript.

### Features
- Glass / neumorphism UI with soft shadows
- Interactive project gallery (images + video)
- Lightbox modal for media viewing
- Subtle parallax hero and scroll-based animations
- Fully responsive layout

### Tech Stack
HTML • CSS • JavaScript

Open `index.html` directly
