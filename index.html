Sarthi - A Crowd Sourced Public Safety WebApplication

Sarthi is an algorithm-driven public safety web application that gathers crowd-sourced safety audits and converts them into meaningful safety insights. The system uses a machine-learning model to predict safety scores and applies geospatial grid encoding with time-decay weighting to prioritize recent incidents. Built with Python and Flask, the backend provides secure authentication using Firebase and exposes REST APIs for heatmap generation and safe-route computation. The frontend integrates authenticated JavaScript modules to deliver interactive safety visualizations.
